#Adapter
Wzorzec używany gdy wykorzystanie instniejącej klasy jest niemożliwe ze względu na jej niekompatybilny interfejs. 
Zwykle adapter jest wykorzystywany w przypadku gdy korzystamy z jakiejś zewnętrznej biblioteki, której użycie jest
problematyczne, np. metody zwracają typy które nie są zgodne z modelem aplikacji. Adapter idealnie wtedy nadaje się do
opakowania takich metod aby zwracały typy zgodne z modelem.
